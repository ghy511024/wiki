兼容的html5 video插件

能找到一款兼容ie7 的html video 插件实属不易，官网或者github 上的最新版都不能做到很好的兼容。

特此记录一下，以备后时之需要 


     <head>
         <link rel="stylesheet" href="http://static.ws.kukuplay.com/common/lib/video/video2-js.css"/>
         <script  src="http://static.ws.kukuplay.com/common/lib/video/v5.0/video2.js"> </script>
         <script  >
            videojs.options.flash.swf = "http://static.ws.kukuplay.com/common/lib/video/v5.0/video-js.swf";
         </script>
     </head>
     <body>
         <video id="h5video" class="video-js vjs-default-skin"   autoplay="true"   data-setup="{}">  
             <source src="http://static.ws.kukuplay.com/movie/fitness.mp4" type='video/mp4'/>  
         </video> 
     </body>

备注：js 和css 版本请不要随意更改，一改绝对不好使（ie7,8），尽量类库引用放在头部，flash 赋值部分放在头部。

自动播放代码


                    var myPlayer = videojs('h5video');
                    videojs("h5video").ready(function () {
                        var myPlayer = this;
                        myPlayer.play();
                    });
