ajax 异步上传图片 样例

传统图片通过表单提交，会引起页面刷新，jquery ajaxupload 可以实现通过ajax 上传图片不引起页面刷新

但这个插件在上传接口返回的数据为json格式的数据时，往往会不好使。所以本文中的类库对原来的插件稍加了部分修改，使得更为通用。

http://static.ws.kukuplay.com/common/lib/ajaximg/ajaxupload.js

版本并不是最新版本，但这个版本一直沿用至今，没出啥问题。

1:使用demo



          <script type="text/javascript" src="http://static.ws.kukuplay.com/common/lib/ajaximg/ajaxupload.js">  </script>
      
        $(document).ready(function(){
            new AjaxUpload($("#upimg-btn"), {
                action: '/picture/multiupload',
                name: 'upfile',
                data: ({}),
                autoSubmit: true,
                onSubmit: function (file, ext) {
                },
                onComplete: function (file, ret) {
                // ret 为上传完图片后的返回数据
                // 数据可能是字符串，也可能是数组，也可能是 json
                }
            });
        })

ps：备注 不支持跨域，上传接口地址必须和页面是同域名，（可使用ngix 中转）
