一键复制插件

点击复制代码引用样例


         <script  src="http://static.ws.kukuplay.com/common/lib/jquery/v1.9.2/jquery-1.9.2.js"> </script>
         <script  src="http://static.ws.kukuplay.com/common/lib/zclip/jquery.zclip.min.js"> </script>
         <script >
                $('#xxx').zclip({
                    path: 'http://static.ws.kukuplay.com/common/lib/zclip/ZeroClipboard.swf',
                    copy: function () {
                        return "复制内容"
                    },
                    afterCopy: function () {
                        alert("复制成功")
                    }
                });
         </script>

如果出现 swf 请求被拒绝，可以尝试延时10毫秒后执行 zclip 函数
