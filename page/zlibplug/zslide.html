章鱼轮播插件 （2017 开年第一弹）

1.引入插件


      <script type="text/javascript" src="xxx/zlib.js"></script>
      <script type="text/javascript" src="xxx/zslide.js"></script>
      <link rel="stylesheet" href="xxx/zslide.css"/>

2.code html

html中class类名是必须的，id 和style属性并不是，所以请随意更改id但不要更改class

zslide-wrap 宽高需要用户根据自己的业务需要自己设定


            <div class="zslide-wrap" id="test" style="width:500px;height:300px;">
                <ul class="zslide-content">
                    <li class="zslide-item"  style="background-color: yellow"></li>
                    <li class="zslide-item"  style="background-color: red"></li>
                    <li class="zslide-item"  style="background-color: orange"></li>
                    <li class="zslide-item"  style="background-color: pink"></li>
                </ul>
            </div>

3.使用插件


            <script type="text/javascript">
                var slide1 = new zlib.plug.zslide($("#test"));
            </script>

插件支持ie7,ie8以下转圈动画将失效但功能均正常使用。

4.参数支持


            <script type="text/javascript">
                var slide1 = new zlib.plug.zslide($("#test"),{
                duration: 5000, // 轮播间隙5秒
                antime: 600, // 轮播切换时间
                locktime: 400, // 点击间隔锁，如果不上锁，则连击效果不好，上锁并且按照动画间隔时间来，则连击感觉卡顿等很久，体验不好，所以取一个比较合适的值
                autoplay: true // 自动轮播
                });
            </script>
